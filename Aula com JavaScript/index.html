JavaScript é uma linguagem de programação de alto nível, interpretada e amplamente utilizada para criar páginas web interativas. Ela é uma das três tecnologias fundamentais da web, ao lado de HTML (estrutura) e CSS (estilo). Sua principal função é adicionar dinamismo e interatividade às páginas, como animações, validações de formulários, atualizações de conteúdo sem recarregar a página e muito mais. Criada em 1995 por Brendan Eich, inicialmente chamada de LiveScript, foi renomeada para JavaScript como estratégia de marketing, mas não tem relação direta com a linguagem Java. Desde então, evoluiu significativamente e é padronizada pela especificação ECMAScript, garantindo consistência em diferentes navegadores
